# Test Automation Suite

This directory contains the automated tests for the React AWS EC2 Nginx application.

## Directory Structure

- `features/` - Feature-based test organization
  - `login/` - Login feature tests
  - `navigation/` - Navigation feature tests
  - `home/` - Home page feature tests
- `fixtures/` - Static JSON test data files
- `page-objects/` - Page Object Model implementation
  - `base-page.ts` - Base page class
  - `pages/` - Page classes
  - `components/` - Reusable component classes
- `helpers/` - Helper utilities
